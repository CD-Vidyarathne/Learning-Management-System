This app is basic class management app.

There are 3 roles.

1. Student
2. Teacher
3. Admin

Anybody who registering into the system first registered as a normal user/student. Then if they want they can apply for the teacher role. If the person satisfy the requirements the Admin can approve them as teachers.

Teachers can create classes. Classes have date time and external-link to join. Students can join classes from their end.
